## 技术要点速查

### 🔧 核心技术栈
- **后端**: Go 1.23 + Wails v2 + go.bug.st/serial
- **前端**: React 18 + TypeScript + Chakra UI v3
- **协议**: Modbus RTU + 自研协议栈
- **构建**: Vite + Go Build + Wails CLI

### 📊 关键数据流
```
DDSU666 设备 ←→ RS485 ←→ 串口通信层 ←→ Modbus协议层 ←→ 数据轮询器 ←→ 业务服务层 ←→ Wails桥接 ←→ React前端
```

### 🎛️ 主要配置
- **串口**: 9600 baud, 8N1, Modbus RTU
- **轮询**: 电参量 1秒, 电能 10秒
- **重试**: 最多 3次, 间隔 50ms
- **超时**: 200ms 读取超时

### 📁 项目结构
```
DDSUViewer/
├── docs/steering/     # 技术文档
├── frontend/          # React 前端
├── internal/          # Go 后端包
├── main.go           # 程序入口
├── app.go            # Wails 应用层
└── wails.json        # Wails 配置
```
